# Calkulater

Статический Astro-сайт с 22 онлайн-калькуляторами для финансов, валют, спорта, ремонта и работы с датами.

## Что внутри

- Astro + TypeScript + Tailwind CSS.
- React Islands для интерактивных форм калькуляторов.
- Динамические страницы из `frontend/src/data/calculators.ts`.
- SEO-каркас: meta-теги, canonical, Open Graph, sitemap, robots.txt, FAQPage, BreadcrumbList, WebApplication, HowTo и Article JSON-LD.
- Интерактивный каталог `/calculators/` с поиском, быстрыми запросами, сортировкой, фильтром по категориям и no-JS fallback.
- Копирование ссылки на расчёт, копирование результата и печать/сохранение результата в PDF.
- Улучшенные карточки калькуляторов: популярность, ключевые темы, количество полей и FAQ.
- Посадочные страницы категорий с быстрыми ссылками, SEO-темами и перелинковкой.
- Сервисные страницы `/about/`, `/contacts/`, `/privacy/` для доверия и базовой юридической упаковки.
- Web manifest, favicon и OG-изображение для базовой упаковки сайта.
- Заготовки под аналитику и рекламные блоки, выключенные по умолчанию.

## Запуск локально

```bash
cd frontend
npm install
npm run dev
```

Сайт откроется на `http://localhost:3000`. Если порт занят, Astro автоматически предложит следующий свободный порт, например `http://localhost:3001` или `http://localhost:3002`.

Если нужен заранее понятный локальный порт без конфликта с основным dev-сервером:

```bash
npm run dev:local
```

Эта команда открывает dev-сервер на `http://127.0.0.1:4321`.

Если Windows только что установил Node.js и команда `npm` не находится, закройте и снова откройте терминал. В крайнем случае можно запускать через полный путь:

```powershell
cd frontend
& 'C:\Program Files\nodejs\npm.cmd' run dev
```

Если нужно быстро проверить уже запущенный локальный сайт, откройте:

- главная: `http://localhost:3000/`;
- каталог: `http://localhost:3000/calculators/`;
- пример калькулятора: `http://localhost:3000/finance/vat-calculator/`.

Чтобы проверить именно собранную статическую версию перед деплоем:

```bash
npm run build
npm run preview:local
```

Preview откроется на `http://127.0.0.1:4322`.

## Проверки

```bash
cd frontend
npm run test
npm run build
```

Или одной командой:

```bash
cd frontend
npm run check
```

`npm run check` запускает unit-тесты, проверку публичных ассетов (favicon, manifest, OG), статическую сборку, проверку битой кириллицы, render-артефактов в HTML, внутренние ссылки в `dist`, SEO-метаданные, покрытие sitemap, служебные hosting-файлы, базовую accessibility-проверку HTML, production-hygiene и performance budget.

## Переменные окружения

В `frontend/.env.example` лежит безопасный шаблон. Для локальной разработки можно ничего не заполнять.

Минимум перед деплоем:

```env
PUBLIC_SITE_URL=https://your-domain.example
PUBLIC_CONTACT_EMAIL=contact@your-domain.example
```

Без `PUBLIC_SITE_URL` локальная сборка использует `http://localhost:3000`, чтобы canonical, Open Graph и sitemap были корректны для теста. Перед публикацией обязательно задайте реальный домен.

Опционально:

```env
PUBLIC_GA_ID=G-XXXXXXXXXX
PUBLIC_YM_ID=12345678
PUBLIC_SHOW_AD_PLACEHOLDERS=true
```

Если ID аналитики пустые или оставлены демонстрационными (`G-XXXXXXXXXX`, `12345678`), скрипты Google Analytics и Яндекс Метрики не подключаются.
Если `PUBLIC_SHOW_AD_PLACEHOLDERS` не равен `true`, пустые рекламные блоки не отображаются.

## Деплой

Проект рассчитан на статический деплой. Подойдут Vercel, Netlify или Cloudflare Pages.

Настройки для хостинга:
- root/build directory: `frontend`;
- build command: `npm run build`;
- output directory: `dist`;
- environment variables: значения из блока выше.

VPS для текущей версии не нужен.

## Перед публикацией

Без домена и монетизации проект уже можно тестировать локально и хранить на GitHub. Перед публичным деплоем останется:

- выбрать домен и задать `PUBLIC_SITE_URL`;
- указать рабочий `PUBLIC_CONTACT_EMAIL`;
- при необходимости подключить `PUBLIC_GA_ID` / `PUBLIC_YM_ID`;
- отдельно проверить актуальность налоговых и валютных правил;
- после подключения рекламы включать `PUBLIC_SHOW_AD_PLACEHOLDERS=true` только для подготовки рекламных мест.

## Важные заметки

- Курсы валют сейчас демонстрационные.
- Налоговые калькуляторы носят справочный характер: перед публикацией и использованием в реальных решениях нужно сверить актуальные правила НДФЛ и НДС.
- НДФЛ и НДС в текущей версии сверены с материалами ФНС России по состоянию на 2026 год:
  - НДФЛ: прогрессивная шкала 13/15/18/20/22% с порогами 2,4 / 5 / 20 / 50 млн ₽ в год;
  - НДС: основная ставка 22% с 1 января 2026 года, также оставлены 20% для старых документов, 10%, 0%, 5% и 7% для отдельных режимов.
- Все расчеты выполняются в браузере; сайт не сохраняет введенные пользователем значения.
